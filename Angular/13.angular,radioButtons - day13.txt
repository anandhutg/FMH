TWO WAY BINDING FOR PROPERTY VIEW :

Add in add property html
<div class="col-4">
  <div class="card-header">
    <h3>Your Property Preview</h3>
  </div>
  <div>
    <app-property-card [property]="propertyView"></app-property-card>
  </div>
</div>

Then in component define propertyView as ->
propertyView :IProperty={
    Id :0,
    Name:'',
    Price:0,
    SellRent:0,
    Type:""
  };
For two way binding of price property in add property html add in <input> -> [(ngModel)]="propertyView.Price"

<input type="text" class="form-control" name="price" [(ngModel)]="propertyView.Price">
