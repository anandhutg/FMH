To retrieve data into angular ->

In housingService.ts define ->  
getAllCities(): Observable<string[]>{
    return this.http.get<string[]>('http://localhost:5289/api/city');
  }

In add-propey.ts ->
Define array cityList!: string[];

To copy api data to array cityList ->

ngOnInit() {
    this.CreateAddPropertyForm();
    this.housingService.getAllCities().subscribe(data =>{
      this.cityList=data;
    });
  }
